# Compass

A personal job intelligence pipeline that scrapes, normalizes, deduplicates, and aggregates job postings from multiple boards into a daily digest.

Built to solve a real problem in my own job search — and to demonstrate production-grade data engineering practices end-to-end.

## Stack

Python 3.10 · Scrapy · Playwright · MongoDB · MinIO · Dagster · Docker

## Status

🚧 v1 in active development. See `PROJECT_SPEC.md` for full scope and acceptance criteria.
